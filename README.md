# Task Management System

Веб-приложение для управления задачами с поддержкой:
- Авторизации пользователей
- Проектов с владельцами
- Задач с настраиваемыми полями
- Системы прав доступа
- Kanban доски
- Фильтрации, сортировки и группировки

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

Приложение будет доступно по адресу: http://localhost:8000

## Тестовые пользователи

- admin / admin123 - администратор
- user1 / user123 - обычный пользователь
